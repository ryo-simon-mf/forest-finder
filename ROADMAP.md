# Roadmap - Forest Finder

## バージョン計画

### v0.1.0 - MVP (Minimum Viable Product)

**目標**: 基本機能の動作確認

```
[環境構築] → [PWA設定] → [GPS取得] → [地図表示] → [現在地マーカー]
```

- Next.js + TypeScript + PWA環境構築
- GPS位置情報取得
- Leafletによる地図表示
- 現在地の地図上表示
- **iOS/Androidでのインストール確認**

---

### v0.2.0 - 森林データ連携

**目標**: 国土地理院データとの連携

```
[API調査] → [データ取得] → [地図上表示]
```

- 国土地理院の植生データAPI調査
- 森林エリアデータの取得実装
- 森林ポリゴンの地図上表示

---

### v0.3.0 - 距離計算機能

**目標**: 核心機能の実装

```
[距離計算] → [最寄り検出] → [UI表示]
```

- Haversine式による距離計算
- 最寄りの森林検出ロジック
- 距離のリアルタイム表示UI

---

### v0.4.0 - モバイルUX最適化

**目標**: iOS/Androidでの体験向上

```
[タッチ操作] → [レスポンシブ] → [パフォーマンス]
```

- モバイルファーストのUI改善
- タッチジェスチャー最適化
- ローディング・エラー表示改善
- ダークモード対応

---

### v0.5.0 - オフライン対応

**目標**: ネットワークなしでも使える

```
[Service Worker] → [キャッシュ戦略] → [フォールバック]
```

- Service Workerによるキャッシュ
- オフライン時のUI
- 地図タイルのキャッシュ

---

### v1.0.0 - 正式リリース

**目標**: 本番運用開始

```
[テスト] → [最適化] → [デプロイ]
```

- 全機能のテスト完了
- Lighthouseスコア90+達成
- Vercelへの本番デプロイ
- **App Store / Play Store への登録（TWA）** ※オプション

---

## 将来的な拡張案

### v1.x - 機能拡張

- [ ] 複数の森林候補表示
- [ ] ナビゲーション機能（ルート案内）
- [ ] お気に入り森林の保存
- [ ] 森林の詳細情報表示（面積、種類など）
- [ ] プッシュ通知（近くに森があるとき）

### v2.x - 高度な機能

- [ ] AR表示（カメラで方向表示）
- [ ] 森林の混雑度予測
- [ ] 季節ごとの見どころ情報
- [ ] コミュニティ機能（レビュー・写真共有）
